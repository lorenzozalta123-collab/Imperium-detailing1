# Imperium Detailing Mobile App

Progetto Expo/React Native pronto per iOS e Android.

## Avvio locale
```bash
npm install
npm start
```

## APK Android con Expo EAS
1. Installa Expo/EAS:
```bash
npm install -g eas-cli
```

2. Accedi:
```bash
eas login
```

3. Genera l'APK:
```bash
eas build -p android --profile preview
```

## Android Studio
Puoi anche usare:
```bash
npx expo run:android
```
